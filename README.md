# Dengue Infection Prediction Using Machine Learning

# Project Overview
Developed a machine learning pipeline in Python to predict dengue infection using clinical biomarkers and environmental factors.

#ML Models Implemented
- Logistic Regression  
- Decision Tree  
- Random Forest  
- Bagging  
- AdaBoost  
- Gradient Boosting  

# Data Preprocessing
- Missing value imputation  
- Feature scaling  
- Categorical encoding using pandas and scikit-learn pipelines 

# Experimental Cases
- Case-1: Environmental + NS1  
- Case-2: Environmental + IgG  
- Case-3: Environmental + IgM  

# Results
- Achieved **100% prediction accuracy** in Case-2 (Environmental + IgG)  
- Above **97% accuracy** in Case-1 (Environmental + NS1)  
- Evaluated using Accuracy, Precision, Recall, F1-score, and MAE  

# Libraries Used
- Python: pandas, scikit-learn
